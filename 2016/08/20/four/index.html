数据类型

####typeof操作符
 ECMAScript是松散类型的，在变量申明时并没有使用与之类型相对应的关键字，如果在代码中想要获知某个变量的基本数据量，就可以使用typeof。这里要注意的是typeof返回的是字符串类型。

类型 含义
undefined 如果这个值未定义
boolean 如果这个值是布尔值
string 如果这个值是字符串
number 如果这个值是数值
object 如果这个值是对象或null
function 这个值是函数 
var testUndefined = undefined;  
var testUndefined1;  
var testNull = null;  
var testObject = {a:1};  
var testFunction = function(){return;};    
console.log(testUndefined);//"undefined"  
console.log(testUndefined1);//"undefined" 
console.log(testNull);//"object"  
console.log(testObject);//"object"  
console.log(testFunction;//"function"

在ECMAScript中函数也是对象，但是函数又要一些区别与其他对象的特定，所以ECMAScript在使用typeof的时候将其区分开来。

####基本数据类型
 1.undefined
 Undefined是只有一个值的数据类型，已经声明但是并未初始化的变量有拥有的默认值就是undefined。字面值undefined的主要目的是用于比较，引入这个值的目的是区分空对象指针与未经初始化的变量。
 未初始化的变量与尚未定义的变量还是有区别的，尚未定义的变量只能够使用typeof操作符检测其数据类型，对并未初始化的变量与尚未定义的变量使用typeof操作符return值相同。

var test1;
//假如有未定义的变量test2
console.log(test1);//"undefined"
console.log(test2);//错误
console.log(typeof(test1)==typeof(test2));//"ture"

 2.Null
Null是只有一个值的数据类型，这个值就是null。
从逻辑角度看，null代表一个空对象指针，这也是typeof操作符检测null值return“object”的原因。
如果定义变量的打算将来用来保存对象，那么最好将该变量初始化为null而不是其他值。这样一来，只要检测null值就可以知道相应的变量是否已经保存了一个对象的引用了。
 3.Boolean
 该类型只有两个字面值：true和false。这两个值与数字值不是一回事，因此true不一定等于1，而false也不一定等于0。在javascript中字母是区分大小写的，所以True和False不是Boolean的值。
　要将一个值转换为其对应的Boolean值，可以调用类型转换函数Boolean()，

alert(Boolean("a"))//true  
//将空字符串转化成false  
alert(Boolean(""))//false  
alert(Boolean({a:"a"})//true
//将对象转化成true


由于布尔值常用与很多判断语句，这些转换规则对理解流控制语句（如if语句）自动执行相应的Boolean转换非常重要。
 4.Number
 这种类型用来表示整数和浮点数值，还有一种特殊的数值，即NaN（非数值 Not a Number）。
 a.整数最基本的数值字面量格式是十进制整数，除此之外还有8进制和16进制整数。

var intnum=18;
var octnum=070;//转换为十进制的56
var octnum1=08;//解析为8
var hexnum=0xA;//转换为十进制的10

在进行计算时，所有以八进制和十六进制表示的数值最终都将被转换成十进制。
 b.要定义浮点数必须包含小数点和小数点之后的一位数字（非0）。因为保存浮点数的内存空间是整数的两倍，所以js会不失时机的将浮点数转换为整数。

var floatnum=1.//解析为1
var floatnum=1.0//解析为1

科学计数法

var floatnum=6.66e4//66600

浮点数在进行算术运算时，其精度远远不如整数，列如0.1+0.2!=0.3,而是0.3000000000004这类的；
浮点字面量有趣之处在于，用它进行计算之前，真正存储的都是字符串。（节约内存呢？）
 c.数值范围
Max：Number.MAX_VALUE 能表示的最大数
Min：Number.MIN_VALUE 能表示的最小数
Number.POSITIVE_INFINITY Infinity Number.NEGATIVE_INFINITY（负） -Infinity
某次计算超出JavaScript的值，这个数值就会转为 Infinity。
d.NaN数值用于表示一个本来要返回数值的操作数未返回数值的情况（这样就不会抛出错误了）。例如，在其他编程语言中，任何数值除以0都会导致错误，从而停止代码执行。但在JavaScript中，任何数值除以0会返回NaN，因此不会影响其他代码的执行。
特点一：任何涉及NaN的操作（NaN/10）都会返回NaN，这个特点在多步计算中可能导致问题。

特点二：其值与任何值都不相等，包括他自己。 NaN == NaN return false；

根据NaN的两个特点，js中定义了isNaN（）函数，这个函数接受一个参数，该参数可以是任何数据类型，函数帮助我们确定该参数是否“不是数值”。在函数接受到参数之后，会尝试将其转换为数值。某些不是数值的值会被直接转换为数值，比如（”10”或者boolean值），任何不能转换为数值的值都会让函数返回 true。

console.log(isNaN(10));//false
console.log(isNaN("10"));//将字符串转换为数值false
console.log(isNaN(blue));//true不能被转换
console.log(isNaN(true));//false能被转换

f.数值转换
3个 函数可以把非数值转换为数值：Number，parseInt（俩参数），parseFloat（只有一个参数，且只能解析十进制）。
Number（）、parseInt( )、parseFloat( )转换规则。

var num1=Number("hello");  //NaN
var num2=Number("");    //0
var num3=Number(true);  //1
var num4=parseInt("123asd") //123
var num5=parseInt("0xA")   //10
var num6=parseInt("0xAF",16)   //175
var num7=parseFloat("0023.5")//23.5
var num8=parseFloat("12") //12

 5.string
 String类型用于表示由零或多个16位Unicode字符组成的字符序列，即字符串。字符串可以由单引号(‘)或双引号(“)表示。
a. 字符字面量
b.字符串特点
   Js中的字符串是不可变的，要改变某个变量的字符串的值，只能销毁原来的字符串的值，然后创建一个新的字符串变量填充该变量。

Var lang = “java”;
Lang = lang + “script”

c.转换为字符串
toString( )（一个可选参数：输出数值的基数）和String( )方法。还可以将要转换的值和字符串（””）相加。Tostring返回相应值的字符串体现，null类型和undefined类型没有toString( )方法。但String( )函数可以将任何类型的值转换为字符串。

var num=11;
var num1=null;
var num3;
console.log(num.toString(2));//1010
console.log(num.toString());//"11"
console.log(String(num1));//null
console.log(String(num3));//undefined
